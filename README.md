# AngularJs Project Template

为方便做ng的开发所作的项目模板。

## 运行必备环境及工具安装:

* Node.js
  - gulp
  - bower
* Ruby
  - compass

## gulpfile.js功能配置

* jsonlint 配置文件格式检查
* jshint js脚本检查
* sass 编译scss到css
* images 图片压缩
* html 合并css及js文件并压缩
* tmpl2js 将ng的view打包成js模板并压缩
* copy 将一些有用附件复制到dist目录
* zip 将生成好德dist目录打成压缩包
* clean 清空dist目录
* connect 配置内置http server
* server 启动server
* watch 设置监听文件及采用相应的处理
